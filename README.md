# 💱 Currency Converter

A sleek and responsive currency converter app built with **React** and **Tailwind CSS**.  
It helps users convert between different currencies in real-time using a simple and intuitive interface.

![screenshot](./src/assets/screenshot.png)  <!-- Add actual screenshot path -->

---

## 🚀 Features

- 🔁 Swap between currencies
- 🔢 Live amount conversion
- 🌍 Supports multiple currencies (based on exchange rates API)
- 💡 Simple, clean, and mobile-friendly UI
- 🎨 Styled with Tailwind CSS
- 🧊 Glassmorphism + background image + dark overlay

---

## 🛠 Tech Stack

- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)
- Custom React Hooks (`useCurrencyInfo`)

## 📦 Installation

```bash
git clone https://github.com/your-username/currency-converter.git
cd currency-converter
npm install
```

### Start Development Server

```bash
npm run dev
```

**The app should now be running at:**
**👉 http://localhost:5173**
